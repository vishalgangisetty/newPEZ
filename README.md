# pharmEZ

pharmEZ is a comprehensive medical management web application built with Flask. It allows users to:
- **Manage Medications**: Track prescriptions and receive highly customizable reminders.
- **Find Pharmacies**: Locate nearby pharmacies using GPS or text-based search.
- **Analyze Prescriptions**: Upload and analyze prescription documents using AI.
- **Safety Checks**: Ensure medication safety with interaction checks.
